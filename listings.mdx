Each listing in your directory can contain rich information about the business, resource, or item. Here's a breakdown of all the available fields:

### Basic Information

- **Name** (Required) - The title of your listing
- **Description** - A detailed description of the listing
- **Categories** - Organize listings by selecting or creating categories
- **Tags** - Add relevant keywords to make listings more discoverable
- **Content** - Rich markdown content area where you can:
  - Write formatted text
  - Add images and videos
  - Embed YouTube/Vimeo videos

### Contact Details

- **Phone Number** - Business or contact phone (e.g., +1234567890)
- **Email** - Contact email address
- **Address** - Physical location of the business

### Web Presence

- **URL** - Website address with auto-fill capability
- **Social Links** - Add multiple social media profiles:
  - Choose platform (Twitter, Facebook, LinkedIn, etc.)
  - Add profile URL
  - Easily reorder or remove links

### Media

- **Logo** - Upload a logo image or provide a URL
- **Cover Image** - Upload a header image or provide a URL
- **Screenshot** - Automatically capture website screenshot if no cover image is provided

<Warning>
  To do listing screenshot, you need to enable ScreenshotOne integration in your integrations settings.

  Learn more about [ScreenshotOne integration](/essentials/integrations#screenshotone).
</Warning>

### Visibility Settings

- **Published** - Toggle listing visibility
- **Featured** - Highlight important listings on homepage
- **No Follow** - SEO setting to control search engine behavior for outbound links

## Quick Tips

1. Use the "Get info from URL" feature to automatically populate listing details
2. Organize listings effectively with categories and tags
3. Utilize markdown in the content area for rich formatting
4. Keep contact information up to date
5. Use high-quality images for logos and covers

<Note>
  Featured listings require additional configuration in Directory Settings to control their display behavior.
</Note>
