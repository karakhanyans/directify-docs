## Directories

### Get All Directories

Retrieve all directories owned by the authenticated user.

```http
GET /api/directories
```

**Response:**

```json
{
  "data": [
    {
      "id": 1,
      "name": "Tech Startups Directory"
    },
    {
      "id": 2,
      "name": "Design Tools Directory"
    }
  ]
}
```
