Directify supports any analytics platform that uses a JavaScript snippet. Simply paste your tracking code in the analytics field in your directory settings.

<Frame>
  <img src="/images/analytics.png" alt="Analytics integration in Directify" />
</Frame>

## Examples

### Google Analytics 4
```html
<!-- Google tag (gtag.js) -->
<script async src="https://www.googletagmanager.com/gtag/js?id=GA_TRACKING_ID"></script>
<script>
  window.dataLayer = window.dataLayer || [];
  function gtag(){dataLayer.push(arguments);}
  gtag('js', new Date());
  gtag('config', 'GA_TRACKING_ID');
</script>
```

### Plausible
```html
<script defer data-domain='yourdomain.app' src='https://plausible.io/js/script.js'></script>
```

### [DataFast](https://datafa.st/)
```html
<script
  defer
  data-website-id="YOUR_WEBSITE_ID"
  data-domain="yourdomain.app"
  src="https://datafa.st/js/script.js">
</script>
```

You can add multiple analytics scripts - just paste them all in the analytics field.
