## Step 1: Add Your Domain

1. Go to your **Directories** list
2. Click **"Click to set custom domain"**
3. Enter your domain name

<Frame>
  <img src="/images/set-custom-domain.png" alt="Adding custom domain in Directify" />
</Frame>

## Step 2: Configure DNS

### For Root Domains (A Record)
```
Type: A
Name: @ (or blank)
Value: 159.223.235.17
TTL: Auto
```

### For Subdomains (CNAME Record)
```
Type: CNAME
Name: directory (or your chosen subdomain)
Value: your-directory.directify.app
TTL: Auto
```

<Frame>
  <img src="/images/set-dns.png" alt="DNS record configuration example" />
</Frame>

<Warning>
  If using Cloudflare, set proxy status to **DNS only** (gray cloud). Orange cloud proxying will prevent SSL certificate generation.
</Warning>

## Step 3: Wait for Verification

DNS propagation typically takes:
- **5-60 minutes** - Cloudflare, AWS
- **1-4 hours** - Google, Namecheap
- **12-48 hours** - Some registrars

<Frame>
  <img src="/images/verify-domain.png" alt="Domain verification process" />
</Frame>

## Step 4: SSL Certificate

Once DNS is verified, Directify automatically provisions a free SSL certificate. No action required - HTTPS is enabled automatically.

## Verification Complete

<Frame>
  <img src="/images/verified.png" alt="Successfully verified custom domain" />
</Frame>

## Troubleshooting

**DNS not propagating?**
- Verify DNS records are correct
- Clear local DNS cache
- Check at [whatsmydns.net](https://whatsmydns.net)

**SSL certificate not working?**
- Wait up to 1 hour after DNS verification
- Ensure Cloudflare proxy is disabled (gray cloud)
